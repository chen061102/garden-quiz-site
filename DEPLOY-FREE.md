# 免费部署说明

如果你不想买域名、不想买服务器，这套刷题站可以直接走免费静态托管。

## 最省事方案

1. GitHub Pages
2. Cloudflare Pages

## 先说结论

- 这两个方案都可以免费上线。
- 都不需要你买服务器。
- 都可以直接给你一个公网链接。
- 但是如果你要求“中国大陆长期稳定、速度稳定、微信里始终丝滑”，免费海外托管不能百分百保证。

## 我对你的建议

如果你现在只是想先把网站发出去给自己和朋友用：

先上 GitHub Pages。

原因：

- 免费。
- 步骤最少。
- 不需要绑卡买资源。
- 这套站点本来就是纯静态，和 GitHub Pages 很适配。

## 适合你的预期

- 要的是“先有个网站链接”：选 GitHub Pages。
- 要的是“大陆更稳一些，但仍然免费”：可以再试 Cloudflare Pages。
- 要的是“国内长期稳定商用”：还是得回到备案 + 国内云。
